## Usage
### 1. install dependency
```python
pip install -r requirements.txt
```
### 2. fill information
- 1. edit the [config.json](config.json)
- 2. run program below to get the information of detail address. 
    ```python
    python getaddress.py
    ```

```python
python main.py
```
